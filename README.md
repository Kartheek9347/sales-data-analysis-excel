Sales Data Analysis Dashboard using Excel

This project analyzes sales data to generate business insights and performance reports.

Tasks Performed:
- Data collection and organization
- Data cleaning and validation
- Revenue calculation using Excel formulas
- Data analysis using Pivot Tables
- Dashboard creation using charts
- MIS reporting for management insights

Tools Used:
Microsoft Excel
Pivot Tables
Charts
Basic Excel formulas

Insights Generated:
- Revenue by region
- Category-wise sales performance
- Salesperson performance comparison
